# cat_or_dog_app

This little app classifies images you upload as a cat or  a dog.
The frontend has been done using streamlit.
The model, saved in the repository as model.h5, has been trained before.
The method and the code used to train the model is presented in another repository.

Drag and drop your image to the upload area and press the button.
The app will deliver the prediction estimate.
A value close to zero means the image represents a cat, and a value close to one for a dog.
